Atomic Layer Decomposition Tool

Process Description: 

  • Atomic Layer Deposition (ALD) is a specialized thin-film deposition technique used in semiconductor manufacturing that allows for precise, atomic-level control over the thickness and composition of films. 

  • ALD builds up films one atomic layer at a time through a self-limiting process using a combination of multiple sources of precursors and plasma gases.

 

System Components: 

Vacuum pump: Pumps down and maintains ALD process chamber under high vacuum. 

Precursor lines: Used to flow precursors from their storage containers to the deposition chamber.

Laptop: Displays software used to program, process recipes, startup/shutdown and configure system configuration (e.g., control of the chamber door, gas flow, chamber pressure, chamber temperature, etc.).

Process screen display: Display of main control buttons, recipe table, program tab, gas flow control, and temperature control, and real-time pressure of the process chamber.

Chamber door: Used for loading/unloading samples and keeping the ALD process chamber under high vacuum.

ALD process chamber: Vacuum chamber in which ALD process takes place.

Emergency stop button: In case of an emergency, push the emergency stop button. If pressed, the system operation will halt immediately.
 

Operation Procedure:

  1. Open the software on PC
  2. Verify system status
  3. Run pre-clean and conditioning recipe
  4. Vent the chamber
  5. When the loadlock is fully vented, open the chamber by turning the switch, then press the 2 green button at the same time and hold until the chamber rises and rotates 90 degrees.
  6. Load samples
  7. Close the chamber
  8. Return to computer screen, go to the recipe screen to edit the recipe for etching process (e.g., gases, flow, pressure, time, ...)
  9. Run the desired recipe: click ‘run’ to start the process
  10. After finishing the etching process, vent and open the chamber, then take out your sample.
  11. Pump down the chamber after use.
     

Safety Warnings:

  • Always leave the chamber in vacuum state.
  • Adhere to the list of allowed materials that can be introduced to ICP-RIE
  • Push the emergency stop button in case of emergency.
